# webpack-develop-env

探索 webpack 相关开发配置：HMR 热更新、devtool(source-map)、devServer 等

# 相关命令

```
* `npm run build` 命令，用于单种devtool配置的打包；
* `npm run build-source` 命令，用于多种devtool配置的打包，快速对比不同配置的效果；
* `npm run dev` 命令，用于启动本地服务，验证模块热替换；
```
